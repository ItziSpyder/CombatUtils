# CombatUtils
### Adds combat "utils" for your Minecraft Server.
-----------------------------------------------------
## Combat & Utilities
- Anchor Aura
- Crystal Aura
- Kill Aura
- Reach
- Flight
- Speed
- Auto Totem
- Anti Kb
- Dodge
- No Fall
- Criticals

## Commands
- /#toggle <module>
